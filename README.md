# Git-pract-1
# Перевірка
# Практична робота: Робота з віддаленим Git-репозиторієм

Цей проект присвячений вивченню основ роботи з системою контролю версій Git та платформою GitHub у середовищі Visual Studio.

## Основний функціонал та виконані завдання
- **Ініціалізація репозиторію:** Створення локального Git-репозиторію та його ініціалізація у Visual Studio.
- **Робота з віддаленим репозиторієм:** Підключення проекту до віддаленого репозиторію на GitHub (Push/Pull).
- **Система комітів:** 
- **Синхронізація:** Налаштування та демонстрація регулярної синхронізації локального коду з хмарним сховищем GitHub.
- **Документування:** Створення файлу `README.md` з використанням розмітки Markdown для опису проекту та ходу виконання роботи.

## Процес виконання (Git Workflow)
Для кожної зміни у коді було виконано наступний алгоритм:
1. Внесення змін у вихідний код проекту.
2. Перевірка змін через панель `Git Changes` у Visual Studio.
3. Фіксація змін (Commit) з назвою, що пояснює суть виконаної дії.
4. Відправка змін на сервер (Push) для синхронізації з віддаленим репозиторієм.
